All of these features can be found/copied from https://bootswatch.com/pulse/. If you "inspect" these elements in a browser, you can see the html elements that give them their color. Below I've copied and offered a slightly more detailed explanation of commonly used elements. Please feel free to contact me if you have questions on how to do something.

Bootstrap divides the page into 12 segments. The class "col-lg-#" will tell you how much of the page the item should take up. The number can be any whole number between 1 and 12. Examples are below.
Full Row - Class="col-lg-12"
Half of Page - Class="col-lg-6"
Third of Page - Class="col-lg-4"
Small - Class="col-lg-1"

These are built in colors from the bootswatch theme. To change color, use "-primary", "-secondary" etc
